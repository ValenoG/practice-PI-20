# Задание 4

Вариант 5.
Даны два числа в 200 знаков. Определить большее из них.
Например,
1. Вход:

> 123456789 и 123345678

Выход:
> 123456789 больше, чем 123345678

2. Вход:

> 123456789 и 923345678

Выход:
> 923345678 больше, чем 123456789

3. Вход:

> 123456789 и 123456789

Выход:
> 123456789 == 123456789
