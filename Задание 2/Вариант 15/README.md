# Задание 2


Вариант 15.
Дана структура Время, состоит из трех полей целого типа (Часы, Минуты и Секунды). Необходимо написать функцию, прибавляющую к заданному времени количество минут, 
введенное пользователем (до 1440 минут (1 сутки)).
Например,
1. Вход:

> 12:12:12
>> 4

Выход:

> 12:16:13

2. Вход:

> 12:12:12
>> 65

Выход:

> 13:17:12

3. Вход:

> 12:12:12
>> 1439

Выход:

> 12:11:12
