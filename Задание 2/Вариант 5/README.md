# Задание 2

Вариант 5.
Дана структура Время, состоит из трех полей целого типа (Часы, Минуты и Секунды). Необходимо написать функцию, прибавляющую одну секунду к заданному времени.
Например,
1. Вход:

> 12:12:12

Выход:

> 12:12:13

2. Вход:

> 12:12:59

Выход:

> 12:13:0

3. Вход:

> 23:59:59

Выход:

> 0:0:0
