# Задание 1

Вариант 5. 
Написать программу, которая считывает текст из файла и определяет количество строк, содержащих хотя бы одну из арифметических операций (+, -, *, /).
Например,
1) Вход:

>qwerty

>123 + 5 - 14 / 2

>4 + 5

>abc

>a + b


Выход:

>3



2) Вход:

>3 - 1

>2 * 2 = 4

>999 / 3


Выход:

>3



3) Вход:

>qwe1

>rty

>abc2


Выход:

>0
