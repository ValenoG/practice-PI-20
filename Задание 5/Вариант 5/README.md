# Задание 5

Вариант 5.
Написать функцию поиска студента Иванова в дереве. Если такого студента нет - вывести сообщение.
Например,
1. Список:
- .. Петров ...
- ... Иванов ...
- ... Петрова ...

 Результат выполнения функции:
 
 > В дереве есть студент Иванов.
 
 2. Список:
- .. Петров ...
- ... Иванов ...
- ... Иванов ...

 Результат выполнения функции:
 
 > В дереве есть студент Иванов.

3. Список:
- .. Петров ...
- ... Сидорова ...
- ... Петрова ...

 Результат выполнения функции:
 
 > В дереве нет студента Иванова.
